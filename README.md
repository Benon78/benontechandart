## Getting Started

- Clone the repository

```bash
git clone https://github.com/Benon78/benontechandart.git
cd Benon_techart
```

- Install dependencies

```bash
npm install
```

- Run development server

```bash
npm run dev
```

---

## Environment variables

Create a `.env` (do not commit) and add any required keys. Example:

```
VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_ANON_KEY=your-supabase-anon-key
```
